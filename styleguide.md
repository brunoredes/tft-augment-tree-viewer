# Javascript/Typescript
- Use functional programming;
- Use pure functions;
- Use hoisting;
- Use High Order Function;
- Use function composition;
- Use declarative style;
- Use imutability;
- Use first class functions;
- Use recursion where needed;
- Use pipes where needed transformation;
- Avoid any. Prefer unknown when type is uncert;


# CSS
- Use RAM Pattern on containers;
- Use flex for children when needed;
- Avoid media query. Prefer container query instead;
- Use rem as relative measurements.

# Svelte
- Usedumb/smart components pattern
- always put onMount and onDestroy at the bottom of the script
```
<script lang="ts">
- import statements
- export variables
- local variables
- onMount()
- onDestroy()
- other functions
- reactive statements ($: if(someVariables){})
</script>

- HTML/Svelte code

- <svelte:head> styles

- <style>
```
